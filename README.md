## 🤔 Bu ne?

> *"Dikkat dağınık, kafan karışık, ne iş yapsan yarım mı kalıyor?"*

**Pomodoro App** tam da bunun için! 🍅 İtalyan Francesco Cirillo'nun icat ettiği efsane **Pomodoro Tekniği**ni glassmorphism tasarım + smooth animasyonlarla getirdik masaüstüne.

```
🍅 25 dk Çalış  →  ☕ 5 dk Mola  →  🔄 Tekrarla  →  🏆 4 tur sonra 15 dk büyük mola
```

## ✨ Özellikler

### 🍅 Temel Şeyler

| Özellik | Açıklama |
|---------|----------|
| ⏱️ **Pomodoro Sayacı** | 25/5/15 dk döngüsü, her 4 turda büyük mola |
| 🌙 **Dark / Light Mode** | Tek tıkla, hiç acısız |
| 📋 **Görev Yönetimi** | Ekle, sil, tamamla — basit ve hızlı |
| 📊 **Günlük Hedef** | Kaç pomodoro istiyorsun? Takip et |
| 💬 **Motivasyon Alıntıları** | Her seansta seni çekip kaldıran bir söz |
| ⌨️ **Space Kısayolu** | Mouse'a uzanmadan başlat/durdur |

### 🎵 Ortam Sesleri — Favori Özellik!

> *Kulaklık tak, ses seç, dünya kaybolsun.*

```
🌧️  Yağmur   →  Cafe vibes, sakin ama odaklı
🔥  Şömine   →  Kış gecesi, kitap ve kod
🌲  Orman    →  Doğa sesi, zihin berraklaşır
```

## 🚀 Çalıştır

> *3 komut. O kadar.*

```bash
git clone https://github.com/ziyaguner/pomodoro-app.git
cd pomodoro-app
npm install && npm run dev
```

🌐 Açıl: **`http://localhost:5173`**

Komutları çalıştırdın mı? Tebrikler, artık verimlisin. 🎉

## 📁 Proje Yapısı

```
pomodoro-app/
│
├── 📁 src/
│   ├── 📁 components/
│   │   ├── 🕐 TimerDisplay.jsx       ← Flip animasyonlu büyük sayaç
│   │   ├── 🎮 Controls.jsx           ← Başlat / Durdur / Skip
│   │   ├── ⚙️  SettingsPanel.jsx      ← Her şeyi özelleştir
│   │   ├── 📋 TaskList.jsx           ← Görev yönetimi
│   │   ├── 📊 Stats.jsx              ← Bugün ne kadar çalıştın?
│   │   ├── 🕰️  Clock.jsx              ← Gerçek zamanlı saat
│   │   ├── 🎵 AmbientSound.jsx       ← Yağmur, şömine, orman
│   │   ├── 🌙 ThemeToggle.jsx        ← Dark/Light geçişi
│   │   └── 💬 MotivationalQuotes.jsx ← "Yapabilirsin!" enerjisi
│   ├── 📁 hooks/                     ← Custom hook'lar
│   ├── App.jsx
│   └── main.jsx
│
└── 📁 screenshots/                   ← README için güzel görseller
```
