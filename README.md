# 🍅 Modern Pomodoro App

Bu modern Pomodoro uygulaması, odaklanmanızı artırmak için tasarlanmış şık ve etkileşimli bir zamanlayıcıdır. Glassmorphism tasarım dili, pürüzsüz animasyonlar ve özelleştirilebilir ayarlarıyla verimliliğinizi artırırken göz yormayan, premium bir kullanıcı deneyimi sunar.

Projenizi yerelde çalıştırmak için aşağıdaki komutları kullanabilirsiniz:

```bash
npm install
```
